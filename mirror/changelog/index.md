# Forge changelog

As [previously announced](https://developer.atlassian.com/platform/forge/changelog/#CHANGE-3302 "https://developer.atlassian.com/platform/forge/changelog/#CHANGE-3302"), from Jun 29, 2026, UI Modifications will **not** run for anonymous, unlicensed or customer account users unless the module explicitly declares support for these user types in the Forge app manifest.
