# feedback

## Description

let us know what you think about Forge

## Usage

```
1
Usage: forge feedback [options]
```

## Options

```
1
2
--verbose   enable verbose mode
-h, --help  display help for command
```

## Operation

The `forge feedback` command displays a message that suggests URLs you can visit to report issues in Forge or ask questions, then prompts you for feedback. It sends the feedback to Atlassian and displays a message confirming that it has been sent.
