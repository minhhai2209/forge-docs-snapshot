# storage entities

## Description

manage indexes for custom entities present in your app

## Usage

```
1
Usage: forge storage entities [options] [command]
```

## Options

```
1
2
--verbose          enable verbose mode
-h, --help         display help for command
```

## Commands

```
1
2
help [command]     display help for command
indexes [options]  manage indexes for custom entities present in your app
```

Manage storage entities for your Forge app with the storage entities command.
