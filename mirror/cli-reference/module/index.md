# module

## Description

[Preview] Commands for managing modules in your app. ⚠️ These commands may
modify your app files (manifest.yml, source files, and package.json).

## Usage

```
1
Usage: forge module [options] [command]
```

## Options

```
1
2
--verbose                    enable verbose mode
-h, --help                   display help for command
```

## Commands

```
1
2
3
4
5
add [options]                [Preview] add a module to your Forge App. ⚠️ This
                             will modify your app files (manifest.yml, source
                             files, and package.json).
list [options]               [Preview] list available module templates
show [options] [moduleType]  [Preview] show details about a module template
```
