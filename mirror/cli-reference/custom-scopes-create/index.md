# custom-scopes create

## Description

Submit custom scopes for creation

## Usage

```
1
Usage: forge custom-scopes create [options]
```

## Options

```
1
2
3
4
5
6
--verbose                        enable verbose mode

-f, --file <file>                Path to YAML file containing custom scope
                                 definitions (default: "custom-scopes.yaml")
-h, --help                       display help for command
```
