# variables

## Description

manage app environment variables

## Usage

```
1
Usage: forge variables [options] [command]
```

## Options

```
1
2
--verbose                    enable verbose mode
-h, --help                   display help for command
```

## Commands

```
1
2
3
4
help [command]               display help for command
list [options]               list the environment variables
set [options] [key] [value]  set an environment variable
unset [options] <key>        remove an environment variable
```

## Operation

The `variables` command lets you manage app-specific environment variables for a specific environment (`development`, `staging` or `production`).
