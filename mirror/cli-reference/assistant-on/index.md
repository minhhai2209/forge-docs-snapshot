# assistant on

## Description

[⚠️ experimental feature] enable AI assistant for error analysis (specify name:
rovo or gemini)
When errors occur during Forge command execution, error details will be sent to
your AI agent to help you understand and resolve issues.

## Usage

```
1
Usage: forge assistant on [options] <name>
```

## Options

```
1
2
--verbose   enable verbose mode
-h, --help  display help for command
```
