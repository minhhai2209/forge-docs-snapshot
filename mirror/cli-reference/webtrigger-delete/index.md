# webtrigger delete

## Description

delete a webtrigger URL

## Usage

```
1
Usage: forge webtrigger delete [options]
```

```

2
3
4
5

                                   Bitbucket)
-e, --environment [environment]    specify the environment (see your default
                                   environment by running forge settings list)
-h, --help                         display help for command
```
