# create

## Description

create an app

## Usage

```
1
Usage: forge create [options] [name]
```

## Options

```
1
2
3
4
5
6
--verbose                                      enable verbose mode
-t, --template <template name>                 specify the template to use
-d, --directory <directory name>               specify the directory to create (uses the template name by default)
-s, --developer-space-id <Developer Space id>  specify the Developer Space id to use
-y, --accept-terms                             automatically accept terms and conditions in non-interactive mode
-h, --help                                     display help for command
```
